04. 变量运行在哪里？CPU or GPU?
05. 生成数据 - tf.random
06. 用用均方误差函数 MSE - tf.keras.losses.mse
07. 浅析Dense层 - tf.keras.layers.Dense
08. 维度变换之reshape
09. 维度变换之expand_dims, squeeze
10. 数组的合并 - tf.concat, tf.stack
11. 数组的广播 - tf.broadcast_to
